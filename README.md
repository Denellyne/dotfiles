# Dotfiles

### This Repository hosts all my configurations and dotfiles

#### Tmux

  A minimal configuration with support for ressurection of sessions

#### Nvim

  Based on NVChad, this configuration changes the UI a bit with Noice plugin and also saves your session for the next time

#### Hyprland

  Minimal configuration with support for Waybar HyprShot Hyprlock and a few things more.

  I use Waybar HyprPaper HyprShot HyprLock Wofi Cava and Mako on my setup to give a clean look
